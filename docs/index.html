MP-BioPath

A Mathematical Tool / Algorithm for performing infrence on biological pathways

View project on GitHub

This tool is Written in Julia to develop develop mathematical models from biological pathways through pairwise interaction files. The model derived from these pathways is non linear and provides a means of describing the effect of biological perterbations from observed nodes onto key outputs of the pathway.
